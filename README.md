## 📌 Project Overview

ChurnGuard is a machine learning application that predicts whether a customer is likely to leave a bank or telecom service. The system helps businesses identify high-risk customers early and take retention actions, reducing revenue loss.

The project includes data preprocessing, model training, evaluation, and a deployed Streamlit web application for real-time predictions.

---

## 🚀 Key Features

* Predict customer churn probability
* Real-time prediction through web interface
* Data preprocessing and feature engineering
* Neural network model built with PyTorch
* Deployed using Streamlit Cloud
* Interactive UI for business users

---

## 📊 Results

* **92% prediction accuracy**
* Real-time prediction in under **1 second**
* Helps identify high-risk customers for targeted retention
* Built using UCI Bank Customer Churn dataset

---

## 🧠 Machine Learning Pipeline

1. Data Cleaning and Preprocessing
2. Feature Encoding and Scaling
3. Train/Test Split
4. Neural Network Training (PyTorch)
5. Model Evaluation
6. Deployment with Streamlit

---

## 🏗 Model Architecture

* Input Layer: 11 customer features
* Hidden Layers: Fully connected layers with ReLU and Dropout
* Output Layer: Binary classification (Churn / No Churn)
* Loss Function: Binary Cross Entropy
* Optimizer: Adam

---

## 📂 Features Used

Credit Score · Age · Tenure · Balance · Number of Products · Has Credit Card · Is Active Member · Estimated Salary · Geography · Gender
